# Genesis Roofing

Local frontend project for Genesis Roofing.

## Project Location

`D:\.codex\websites\genesis-roofing`

## Commands

Install dependencies:

```sh
npm install
```

Start the local development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Expected local dev URL:

`http://127.0.0.1:5175`

## Visual Studio 2026

Open `GenesisRoofing.sln` from the project root in Visual Studio 2026.

Install dependencies:
